# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-10

### Added
- Initial release of PrimerPro Python port
- Complete decodability model with grapheme segmentation
- Buildable word search functionality (TextData and WordList variants)
- Residue search for identifying untaught graphemes
- Minimal pair search for phonics contrasts
- Tone chart and tone pair searches
- Comprehensive test suite with 16 test modules
- Zero external dependencies (Python stdlib only)
- Python 3.13+ support
